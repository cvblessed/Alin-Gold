# Alin Gold - Magazin Online

Proiect ecommerce pentru Alin Gold (Moldova), structurat cu:

- `frontend/` pentru pagini HTML, CSS si JavaScript
- `backend/` pentru API Node.js + Express
- `.vscode/` pentru configurari de lucru in Visual Studio Code

## Rulare rapida

1. Frontend: deschide `frontend/index.html` in browser.
2. Backend:
   - intra in `backend/`
   - ruleaza `npm install`
   - ruleaza `npm run dev`

## Observatii

- Toate imaginile sunt placeholder-uri si se vor inlocui ulterior cu poze reale.
- Limba initiala a site-ului: romana.
